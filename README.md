GLEAPHE RADIO ON DESKTOP UBUNTU 

1 . INSTALL

    pip install -r requirements 

2 . CREATE PLAY BUTTON 

      chmod +x play.py
      python play.py

3 . RUN APPLICATION 

      chmod +x gl.py
      python gl.py
